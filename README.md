# Геометрические фигуры, холст

Программа для вычисления данных о геометрических фигурах.

## Возможности

- Добавление кругов, прямоугольников, треугольников с указываемыми величинами.
- Вычисление периметров, площадей фигур как отдельно, так и суммарно по холсту.
- Взаимодействие с холстом: удаление фигур по индексу, минималистичное отображение знаков фигур в терминале.

## Запуск

cd /путь/к/проекту

python main.py   #win
python3 main.py  #lin

## Структура

shapes.py - описание классов фигур, класса холста.
main.py - пользовательский интерфейс

## Требования

Python 3.6

# Geometric shapes, canvas

A program for calculating data about geometric figures.

## Features

- Add circles, rectangles, triangles with input characteristics.
- Calculate the perimeter and area of individual figures.
- Calculate total perimeter and area for all figures on the canvas. 
- Work with canvas: delete figures by index, print text-based images of figures in the terminal.

## Start

cd /path/project

python main.py   #Windows
python3 main.py  #Linux

## Structure

shapes.py - description of classes of figures, the canvas class.
main.py - user interface.

## Requirements

Python 3.6
